# Pac-Man Game

This is a simple Pac-Man game made using HTML, CSS, and JavaScript. The player controls Pac-Man and collects points while avoiding enemies. The project was created to practice JavaScript concepts, game logic, DOM manipulation, and event handling.

## Features

* Pac-Man movement using keyboard controls
* Score tracking
* Enemy movement
* Game Over and Win conditions
* Simple and interactive user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the `index.html` file in your browser.

## Controls

* ↑ Up Arrow
* ↓ Down Arrow
* ← Left Arrow
* → Right Arrow
